Frequently Asked Questions

Do I have to host with BW2?

No. You are free to host anywhere. Most partners choose our Managed Cloud or Private Cloud Tools License because the cost of automation is significantly cheaper than hiring internal DevOps staff.

Can I install BW2 entirely for free on my own server?

Yes. You are free to manually install the BW2 Application Server, Background Service and Postgres database on your own hardware for $0. However, this "Manual Mode" does not include the BW2 Management Containers. You will be responsible for manual updates, backups, traffic routing, and server maintenance.

Do the Private Cloud Tools license include support?

Yes. The license includes support for the BW2 Software Containers (Cloud/Automation/Management layer). You are responsible for the underlying hardware, operating system, and network connectivity.
