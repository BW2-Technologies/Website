Frequently Asked Questions

Can I install BW2 entirely for free on my own server?

Yes. The Core Platform is free to install. However, manual installation requires you to manage the underlying server, updates, and backups yourself unless you use our Private Cloud Tools.

Is the 'Free Platform' really free?

Yes. You can build, distribute, and run the standard version of your application for unlimited users without paying BW2 a cent. We only charge if you or your client specifically requires the Customization Layer or Managed Hosting.

Do I have to host with BW2?

No. Hosting is completely optional. You can self-host on-premise or use any cloud provider of your choice.

Do the Private Cloud Tools license include support?

Yes. If you choose to license our Private Cloud Tools for self-hosting, support for the container stack is included.
